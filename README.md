# Progress Bar Loader 📊

A simple progress bar with animated loading effect built with HTML, CSS & JavaScript.

## Features
- Animated progress bar from 0% to 50% on page load
- Click button to continue from 50% to 100%
- Smooth transitions and gradient colors
- Responsive design

## Technologies
- HTML5
- CSS3 (Tailwind CSS)
- JavaScript

## How to run
Open  in your browser.
